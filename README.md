# Webscraper de Proveedores

Este proyecto de software es para automatizar el proceso de compra de insumos de una 
institucion o dependencia de gobierno, realizando web scraping en las paginas de los 
diferentes proveedores garantizando asi comprar un articulo, producto o insumo al mejor 
precio.

Debe tener un relacion de proveedores y sus paginas web en categorias como Papeleria, 
Equipo de cómputo, Electricidad, Equipo de Oficina, etc.
